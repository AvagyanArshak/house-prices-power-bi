# 🏠 House Prices Analysis — Power BI Dashboard

## 📊 Project Overview
Interactive dashboard analyzing house prices using the Ames Housing dataset (2,930 houses, 83 features).

## 📁 Files
- `House Prices Analysis.pbix` — Power BI Dashboard
- `housing.csv` — Dataset

## 📄 Dashboard Pages
- **Page 1 — Overview:** KPI cards, Neighborhood prices, Area vs Price, Quality trends
- **Page 2 — Price Analysis:** Slicer, Histogram, Treemap
- **Page 3 — Time Trends:** Sales by year

## 🛠️ Tools Used
- Power BI Desktop
- DAX
- Power Query
